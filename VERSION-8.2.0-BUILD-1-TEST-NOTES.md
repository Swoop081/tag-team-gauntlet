# LEGACY Pro Wrestling 8.2.0 — Build 1

## UI polish
- Career outer vertical red side borders removed globally.
- Feud Origin and Match Broadcast primary yellow buttons centred.
- Monday Night Mayhem logo no longer displays LPW; day and show name are retained.

## Match Psychology 2.0
- Every player decision resolves as Major Success, Success, Mixed Result, Failure, or Major Failure.
- Each result immediately changes Match Score, Control, and Crowd.
- A compact result overlay appears before the match resumes.
- Commentary explains the outcome.
- Post-match summary includes a decision-by-decision psychology breakdown.
- Decision cards remain visually neutral with no favourable/balanced/risky colour guidance.
