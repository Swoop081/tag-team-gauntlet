# LEGACY Pro Wrestling

Current web build: **8.6.20**

GitHub Pages repository: `legacy-pro-wrestling`  
Published URL: `https://swoop081.github.io/legacy-pro-wrestling/`

## Deployment

Upload the contents of this folder to the root of the `legacy-pro-wrestling` repository. Keep `.nojekyll` in the repository root. In GitHub Pages settings, publish from the branch/folder used for the live site.

After deploying this version, remove the old iPhone Home Screen shortcut and add the site again so iOS loads the new LEGACY Pro Wrestling icon.

## Runtime files

The live game uses `index.html`, `styles.css`, `game.js`, `data.js`, `update-manager.js`, `service-worker.js`, `version.json`, `manifest.webmanifest`, `.nojekyll`, and the `assets/` directory. Historical test notes and local QA scripts were removed from the release package.
