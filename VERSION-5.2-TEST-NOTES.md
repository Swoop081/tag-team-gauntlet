# Tag Team Gauntlet 5.2 Test Build

## Headline additions
- Gauntlet Cup: eight-team themed, three-round single-elimination tournament flow.
- Achievements screen with persistent unlocks.
- Help & Guide screen.
- Dynamic rivalry recognition after repeated opponent meetings.

## Gameplay tuning
- Decisions have more distinct risk profiles.
- Five-star matches are substantially rarer.
- Finish types have wider score thresholds.
- Singles matches can no longer use the Tag Team Showcase story.
- Opening Classic match remains protected, but is no longer an absolute guaranteed win.
- Crowd starts at zero and must build during the match.

## NPC correction
- Preston Cole is now The Strategist and a manager only.
- His coaching/training event is unassigned and removed from the active event rotation.

## World presentation
- Player-facing faction references removed from wrestler intros and active match history.
- Your Call tab overlap corrected with dedicated spacing.

## Tournament test path
1. Main Menu → Gauntlet Cup.
2. Select exactly two wrestlers.
3. Win Quarterfinal, Semifinal and Final.
4. Confirm championship achievement unlock.
