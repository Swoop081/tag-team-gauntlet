# Tag Team Gauntlet 4.1 — Layout Stability Test

## Included
- Added Hollowman full, portrait and victory artwork to Character Image Manager.
- Between-match cinematic cards now use dedicated portrait artwork for upgraded wrestlers.
- Portrait cards have fixed contained frames so heads and bodies cannot overflow or be chopped by full-art scaling.
- Mobile versus screens stack cleanly on narrow phones instead of squeezing four cards into one row.
- Control Decision overlay now uses an opaque blurred backdrop so wrestler art cannot dominate behind the controls.
- Match portrait containment from 2.1 remains active.
- Cache versions bumped to 4.1.
