# Tag Team Gauntlet 6.0 — Gauntlet Live Build 1

## Included
- Real Gauntlet Live main-menu entry
- Four-founder selection: Jack Mercer, Mason Marks, El Rey del Cielo, Max Justice
- Persistent localStorage career
- Seven-day calendar
- Every fourth Sunday labelled Supercard
- Day-by-day progression
- Resume and new-career controls

## Deliberately deferred to Build 2
- Match-engine integration
- Random events
- Stable recruitment
- Collection locking/unlocking
- Full Supercard gameplay

## Test path
1. Open the game and select Gauntlet Live.
2. Choose one founder.
3. Advance through all seven days.
4. Continue through Week 4 and confirm Sunday is a Supercard.
5. Refresh the browser and confirm the career resumes.
6. Select New Career and confirm replacement of the old save.
